服务条款

最近更新：2026-05-07

当你访问或使用 NODA.PICS（noda.pics，下称"本站"）时，即视为接受以下条款。如不同意，请不要使用本站。

1. 账户

使用 Google / GitHub OAuth 注册账户即可。一个自然人原则上只允许注册一个账户。账户信息（特别是邮箱）须真实有效，否则可能影响付费、密码重置、安全通知。

2. 你提交的内容

你保留对你输入的提示词和生成的图像的全部所有权。我们不会未经允许将你的私人作品用于宣传、训练或转售。

当你主动选择"分享到公共画廊"时，你授予我们在站内展示该作品的非专属许可，至你撤销分享为止。

3. 使用限制

不得用本站生成或上传：

  • 侵犯他人知识产权（含未授权的真人形象、商标、版权角色）
  • 未成年人的露骨内容（任何形式，零容忍）
  • 用于诈骗、伪造身份、伪造证件的图像
  • 违反法律的内容（含但不限于暴恐、毒品、政治敏感）
  • 系统性滥用（脚本批量请求、绕过限速、贩售账号）

违反任一条，我们有权立即冻结账户并删除相关内容，无需提前通知。

4. 付费

Pro 订阅由 Creem.io 处理，按月计费，自动续期。你随时可以取消，取消后当前周期到期前继续可用，到期后转为免费用户。

退款：因平台 bug 导致无法生成图像而你已付费，请通过联系页申请，我们将按比例退款。其他情况一般不予退款。

5. 服务可用性

本站靠 家用 GPU + Render 免费 tier 运行。我们会尽力保持稳定，但不承诺 99.9% 之类的可用性。GPU 维护、断电、机器升级期间可能中断；中断超过 24 小时的，Pro 用户可申请按天补偿。

本站调用的模型（如 Flux）有各自的许可证，使用生成结果时请遵守对应模型的商业用途条款。

7. 终止

你随时可在账户设置中删除账户。我们也可能在你违反本条款、长期不活跃（>2 年）或服务下线时终止账户。
