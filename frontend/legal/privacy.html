隐私政策

最近更新：2026-05-07

本页说明 NODA.PICS（noda.pics，下称"本站"）在你访问和使用服务时收集哪些数据、用于什么目的、与哪些第三方共享，以及你拥有的相关权利。

1. 我们收集什么

注册账号时，我们记录：

  • 邮箱地址（用于登录与通讯）
  • Google / GitHub OAuth 返回的基本资料（用户名、头像 URL、第三方 user id）
  • 使用量（生成次数、最近一次活跃时间，用于限速）
  • 付费记录（订阅状态、支付时间、付款渠道返回的交易号；不存储完整卡号）

未登录访客我们不收集任何账号信息，只按 IP 限速。

你提交的提示词（prompt）和生成的图像会存储在我们的数据库与 Cloudflare R2 中，用于：

  • 展示在你的"我的图库"
  • 排除滥用
  • 用户公开选择"分享到画廊"时，作为公共内容展示

2. 我们不会做什么

  • 不出售你的个人信息给第三方
  • 不用你提交的提示词训练任何模型
  • 不存储完整支付卡号或安全码（由 Creem.io 处理）
  • 不强制你接受任何 cookie 才能使用基础功能

3. Cookie 使用

  • 会话 cookie（HttpOnly JWT）：登录后用于身份识别，关闭浏览器或登出后失效
  • 第三方 cookie：仅由下方"第三方服务"列出的提供方设置

4. 第三方服务

  • Google / GitHub OAuth — 登录身份验证
  • Cloudflare — DNS、CDN、R2 图像存储；按其隐私政策处理请求数据
  • Creem.io — 订阅支付处理
  • Render — 后端托管平台
  • Google AdSense（如启用）— 可能基于你的浏览历史投放个性化广告，可在 google.com/settings/ads 选择关闭

5. 数据保留

账户信息保留至你删除账户。生成的图像默认保留至账户被删除或你主动删除。删除账户的请求请通过联系页提出，14 天内处理。

6. 你的权利（GDPR / CCPA）

无论你身在何处，你都有权：

  • 查询我们对你保存的所有个人信息
  • 请求更正错误信息
  • 请求删除（"被遗忘权"）
  • 导出你的数据（机器可读格式）
  • 反对个性化广告（若启用）

请通过联系页发起请求，我们 30 天内响应。

7. 儿童隐私

NODA.PICS 不面向 13 岁以下儿童。如果你认为有未成年人在我们这里注册，请联系我们删除。

8. 政策更新

我们可能不定期更新本政策，"最近更新"日期会反映最近一次修订。重要变更会在站内通知。
