联系

最近更新：2026-05-07

NODA.PICS 由一个人维护。最快的联系方式是邮件，通常 2-3 个工作日内回复。

邮箱： hello@noda.pics

请告诉我什么

  • Bug 反馈：请尽量附上你的浏览器、操作系统、复现步骤；如果是图片生成相关，附 prompt 和大致时间。
  • 模型新需求：你希望我们接入哪个模型？（Flux 系列、Kling 等都欢迎建议）
  • 付费 / 订阅问题：付款失败、退款、修改订阅状态。请附 Creem 给你的订单号。
  • 隐私请求：根据 GDPR / CCPA 提出的查询、更正、删除、数据导出请求。请用注册邮箱发起，30 天内响应。
  • 合作 / 商务：API 接入、技术分享、广告投放（暂不接受）。
  • 媒体咨询：欢迎报道开源 AI 图像生成的实践，可聊架构、定价心得等。

不要问的

  • 付费让你的工具上首页 / 出现在画廊（不接受任何付费排名）
  • 给个特殊额度无限免费用（请直接订阅 Pro）
  • 帮你绕开外部模型的内容安全限制

开源贡献

NODA.PICS 是开源项目。代码 issue / PR 请直接走 GitHub 仓库，不必通过邮件。
